Module Bundlers

Un Module bundler es una herramienta que nos permite empaquetar y automatizar una serie de tareas y también gestionar los assets de las dependencias que previamente han sido instaladas con el gestor de paquetes, en este caso con npm.

El uso de Module bundler apunta principalmente a archivos JavaScript y su objetivo es fusionar un archivo y sus diferentes dependencias, teniendo como resultado un solo archivo de salida para optimizar los tiempos de carga de una aplicación.

¿Qué Module bundler elegir ?

A continuación ciertas características de cada una de las herramienta para tener en consideración a la hora de declinarse por una u otra

  • Browserify es la herramienta que abrió paso pero ahora está un poco relegado por la nueva generación.
  • Webpack es la herramienta más configurable pero que también requiere un cierto tiempo de aprendizaje antes de poder tener algo concreto.
  • Rollup es interesante pero es una herramienta que sería útil para la publicación de una librería.
  • Fuse-box para los amantes de TypeScript
  • Parcel Pocas configuraciones, para aquellos que desean que funcione sin tomar la iniciativa y que puedan conformarse con el sistema predeterminado.
